# 🌱 OptiCrop – Crop Recommendation System

## 📌 Project Overview

OptiCrop is a Machine Learning-based web application that recommends the most suitable crop based on soil nutrients and environmental conditions. It helps farmers and agricultural users make informed crop selection decisions, leading to better productivity and efficient use of resources.

The application predicts the best crop using soil parameters such as Nitrogen (N), Phosphorus (P), Potassium (K), Temperature, Humidity, pH, and Rainfall.

## ▶️ How to Run the Project

### Clone the Repository

```bash
git clone https://github.com/Bhavagna2007/OptiCrop.git
```

### Navigate to the Project Folder

```bash
cd OptiCrop
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Flask Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

## 📸 Application Workflow

1. Enter soil nutrient values.
2. Enter environmental parameters.
3. Click **Predict Crop**.
4. The Machine Learning model processes the inputs.
5. The application displays the recommended crop and its description.
