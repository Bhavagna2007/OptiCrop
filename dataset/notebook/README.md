# OptiCrop: Smart Agricultural Production Optimization Engine

## Project Description

OptiCrop is a machine learning-based crop recommendation system that predicts the most suitable crop based on soil nutrients and environmental conditions.

## Input Parameters

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH
* Rainfall

## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Machine Learning Algorithm

Random Forest Classifier

## Model Accuracy

99.31%

## Features

* Crop prediction using soil and weather data
* Web interface using Flask
* Data visualization
* User-friendly interface

## How to Run

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Run the Flask application:

   ```
   python flask_app.py
   ```

3. Open:

   ```
   http://127.0.0.1:5000
   ```
